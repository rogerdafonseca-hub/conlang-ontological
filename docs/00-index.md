### **Layer 3: Nominal Lexicon (15,625 words)**

**[Nominal Lexicon Matrix](../lexicon-and-foreign-words/lexicon-nominal-matrix.md)** 🆕 v0.4.0  

**Revolutionary System: CVCV Matrix**

**Structure:** C₁V₁C₂V₂  
- **C₁** = Domain (25 options) — P=beings, M=places, Y=foods, etc.
- **V₁** = Subcategory (5 options) — Type within domain
- **C₂** = Concept (25 options) — Specific concept
- **V₂** = Variation (5 options) — Variant/gradation

**Examples:**
- `peka` = dog (P-domain, e-mammals, k-domestic, a-first)
- `maka` = house (M-domain, a-dwellings, k-residence, a-standard)
- `yaka` = bread (Y-domain, a-grains, k-baked, a-basic)

**Total capacity:** 15,625 systematic words  
**Expandable to:** 1,953,125 (with CVCVCV)  
**Learn by:** Pattern recognition, not memorization  
**Status:** Framework complete, matrices in development

**This is the breakthrough:** First conlang with fully systematic nominal vocabulary.

### **Layer 3B: Numeral System**

**[Numeral System](../lexicon-and-foreign-words/numeral-system.md)** ⏳ Documented in Structure

**Z-Marker Universal System:**

**za-** = Cardinal (1, 2, 3...)  
**ze-** = Ordinal (1st, 2nd, 3rd...)  
**zi-** = Fractional (1/2, 1/3, 1/4...)  
**zo-** = Multiplicative (2×, 3×, 4×...)  
**zu-** = Collective (pair, trio, quartet...)

**Examples:**  
zane = 1, zanenu = 14, zanenunosi = 1,437

**Status:** Complete system, documented

## 🏗️ System Architecture Overview

### **Four Progressive Layers**

**LAYER 1: DUAL SYSTEM GRAMMAR (250 particles)**  
├── CV (125) — Basic grammar  
└── VC (125) — Extended grammar  
    └── Learn in: 2-4 weeks

**LAYER 2: CORE VOCABULARY (625 words)**  
├── VCV pattern — Vowel + Consonant + Vowel  
├── 25 domains × 25 words  
├── Systematically derived from ontology  
└── Learn in: 3-6 months

**FLUENCY THRESHOLD: 875 resources** (comparable to Basic English: 850)

**LAYER 3: NOMINAL LEXICON (15,625 words)** 🆕 v0.4.0  
├── CVCV matrix — C₁V₁C₂V₂ systematic structure  
├── 25 domains × 5 subcategories × 25 concepts × 5 variations  
├── **ZERO arbitrary vocabulary — 100% pattern-derived**  
├── Expandable to 1,953,125 (CVCVCV if needed)  
└── Learn by: Pattern recognition (not memorization)

**LAYER 3B: NUMERAL SYSTEM**  
├── Z-marker system (za/ze/zi/zo/zu)  
├── All number types (cardinal, ordinal, fractional, etc.)  
└── Positional notation (syllables = magnitude)

**LAYER 4: TECHNICAL VECTORS (390,625 concepts)**  
├── VCV+CV+CV+... pattern (written as one word)  
├── 25 matrices × 5 subcategories × 25 vectors × 5 variations × 5 gradations  
├── Optional — use when precision is needed  
└── Learn by: Reference as needed (like technical dictionary)

**Total systematic resources:** 407,125+ (all pattern-derived)
