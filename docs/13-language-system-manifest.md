# 🌐 LANGUAGE SYSTEM MANIFEST — v0.7.4

## 🧠 CORE PRINCIPLE

The system is built on **pure semantic primitives**.

Each consonant represents a **root ontological domain**:
- irreducible
- highly productive
- cross-combinable with all others

The language is generated through **combinatorial semantics**, not memorization.

---

## 🔒 THE 20 ROOT DOMAINS (NON-NEGOTIABLE)

Each domain MUST satisfy:

- High frequency in real-world cognition
- High combinatorial fertility
- Conceptual independence (non-derivable from others)
- Ability to generate large lexical families

### Fundamental Rule:

> A root domain must generate meaning **in all directions**, not only downward (subcategories).

---

## ⚖️ DOMAIN PURITY RULE

Each domain represents **only ONE ontological type**:

| Type | Description |
|------|------------|
| Entity | Thing that exists (object, state, structure) |
| Process | Something that happens |
| Relation | Connection between entities |
| Field | Abstract measurable or positional space |

### Forbidden:

- Mixing process + object (❌ “communication” as both act and message)
- Mixing structure + instance
- Mixing cause + effect

---

## 🧩 COMPOSITION RULE

All complex meaning is built through **domain combination**, not domain expansion.

### Example:

- “message” → B (communicative artifact)
- “to speak” → V + B
- “conversation” → B + X
- “language” → B + L
- “tool for communication” → D + B

> If a concept can be built through combination, it MUST NOT become a base domain.

---

## 🚫 ANTI-SPECIALIZATION RULE

Domains must NOT represent:

- Cultural categories (e.g. food, housing)
- Specific industries or functions
- Narrow semantic fields

### Example (invalid domains):

- Food ❌ → should be P + N
- Dwelling ❌ → should be L + G + K

---

## 🔁 COMBINATORIAL GENERATION

Meaning is created through:

### Structure:
C + V + C (+ V)

Where:
- First consonant = primary domain
- Second consonant = interacting domain
- Vowels = modulation (gradation, nuance)

---

## 🔤 VOWEL FUNCTION

Vowels encode **gradation or modulation**, typically:

| Vowel | Function (default axis) |
|-------|------------------------|
| a | minimal / absence |
| e | low / weak |
| i | neutral / standard |
| o | strong / marked |
| u | extreme / maximal |

⚠️ Some domains invert or redefine this scale (must be explicitly declared).

---

## 🔗 ROLE OF CONSONANT CLUSTERS (ECs)

Clusters (PR, BR, TR, etc.) are:

- **Secondary semantic operators**
- Derived from root domain logic
- Used when a semantic field becomes:
  - highly recurrent
  - structurally stable
  - not efficiently expressible via simple binary combinations

### Key Rule:

> ECs are NOT shortcuts — they are **stabilized semantic patterns**.

---

## ⚠️ PHONOLOGICAL CONSTRAINTS

- Not all consonants combine freely
- Some are inherently clusters (e.g. Ľ)
- System must respect:
  - pronounceability
  - phonotactic stability
  - clarity between CV, VC, and VCV

---

## 🧪 VALIDATION CRITERIA (BACKTEST RULE)

Every domain must pass:

### 1. Coverage Test
Can it combine with most other domains meaningfully?

### 2. Redundancy Test
Does it duplicate another domain?

### 3. Fertility Test
Can it generate large lexical families?

### 4. Frequency Test
Will it be used often in real communication?

### 5. Clarity Test
Is its meaning immediate and intuitive?

---

## 🧭 SYSTEM ARCHITECTURE

### Layer 1 — Ontological Domains
20 root consonants

### Layer 2 — Combinatorial Fields
C + V + C interactions

### Layer 3 — EC Operators
Stabilized semantic clusters

### Layer 4 — Lexical Mapping
Assignment of phonological forms (CVCV, etc.)

---

## 🧠 FINAL PRINCIPLE

> Do not create meaning by adding categories.
> Create meaning by combining primitives.

This ensures:

- infinite scalability
- minimal memorization
- maximum expressive power
- structural coherence

---

## 🚨 GOLDEN RULE

> If a concept feels “natural” as a domain but fails combinatorial tests,
> it is NOT a domain — it is a **derivation**.

---

## ✅ SYSTEM GOAL

A language where:

- every word is logically traceable
- every combination is meaningful
- every domain is essential
- nothing is arbitrary
