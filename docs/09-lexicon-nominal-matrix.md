# 📘 NOMINAL LEXICON MATRIX

**Ontological Organization and Functional Modifiers**

_Version 0.6.0 — Complete Restructuring_

---

## 🎯 Overview

The nominal lexicon is organized by **ontological domains** using the 20 base consonants + 7 defined consonant clusters. Each domain contains systematically derived words that can be modified using the 24 functional modifiers.

**Core Principle:**
- **CV¹** = Ontological domain (broad category)
- **CV²** = Subcategory (specific type within domain)
- **Vowel gradation** = Systematic variation within subcategory
- **CV(final)** = Functional modifier (ALWAYS last syllable, ONE per word maximum)

---

## 📐 WORD STRUCTURE RULES

### **Basic Structure:**

```
[Domain][Subcategory][...Additional Syllables...][MODIFIER]
  CV¹       CV²              CVⁿ                    CV(final)
```

### **Position of Modifier:**

- **2-syllable base** → modifier is 3rd syllable
- **3-syllable base** → modifier is 4th syllable
- **4-syllable base** → modifier is 5th syllable
- etc.

### **Examples:**

```
paba (water, 2 syllables)
→ pabama (water + magnitude-min) = drop
→ pabamu (water + magnitude-max) = sea

pabaka (collected water, 3 syllables - hypothetical)
→ pabakava (collected water + movement-min) = pond
→ pabakavu (collected water + movement-max) = waterfall
```

## 🎨 COMPREHENSIVE EXAMPLES

### **Example 1: Water Domain (from P - Substance)**

**Base word:** pepi (liquid state, normal density)

**With Magnitude Modifier (M):**
- **pepima** = tiny liquid drop
- **pepime** = small liquid amount (puddle)
- **pepimi** = normal liquid amount
- **pepimo** = large liquid body (lake)
- **pepimu** = huge liquid body (ocean)

**With Quality Modifier (L):**
- **pepila** = terrible quality liquid (toxic)
- **pepile** = low quality liquid (dirty)
- **pepili** = normal quality liquid
- **pepilo** = high quality liquid (clean)
- **pepilu** = perfect quality liquid (pure)

**With Movement Modifier (V):**
- **pepiva** = still liquid (pond)
- **pepive** = slowly moving liquid (creek)
- **pepivi** = normally flowing liquid (stream)
- **pepivo** = fast moving liquid (river)
- **pepivu** = rapidly moving liquid (waterfall)

## 📊 GRAMMATICAL CASES (via VC particles)

Cases are marked by **VC particles AFTER** the noun, not by vowel endings:

| Particle | Case | Example |
|----------|------|---------|
| **pa** | allative (to/toward) | pepima pa = to the drop |
| **pe** | perlative (through) | pepima pe = through the drop |
| **pi** | locative (in/at/on) | pepima pi = in the drop |
| **po** | ablative (from) | pepima po = from the drop |
| **pu** | comitative (with) | pepima pu = with the drop |

## 🎯 DESIGN PRINCIPLES

### **What IS marked:**

✅ **Superlative/Comparative:** fo (more/-er), fu (most/-est) via F family  
✅ **Cases:** Full set via P (preposition) family CV/VC  
✅ **Plural/Number:** N (quantity) family or K modifier  
✅ **Diminutive/Augmentative:** M modifier (magnitude)  
✅ **Tense/Aspect:** T (tense), G (aspect) families  
✅ **Evidentiality:** Ľ family (hearsay → witnessed)  

### **What is NOT marked (intentional gaps):**

❌ **Definite/Indefinite articles:** Use demonstratives (di = this ≈ the)  
❌ **Grammatical gender:** System is genderless by design  
❌ **Animacy marking:** Not marked (optional semantic feature)

## 📋 VOCABULARY STATISTICS

**Current System (CVCV base words):**
- 20 base consonant domains × 25 words = 500 words
- 7 EC domains × 25 words = 175 words
- **Total base vocabulary: 675 CVCV words**

**With modifiers:**
- Each base word × 24 modifiers × 5 vowels = 120 variants per word
- **Theoretical total: 81,000 words** (675 × 120)

**With 3-syllable expansion (CVCVCV):**
- Each 2-syllable base × 25 third syllables = exponential growth
- Estimated potential: 500,000+ words
