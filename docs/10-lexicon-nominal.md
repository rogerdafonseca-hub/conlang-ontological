# 10. Nominal Lexicon

**Derivation Rules and Word Formation System**

_Version 0.8.0 — April 2026_

---

## Overview

The nominal lexicon is the **human interface** of the language — the actual spoken words that people use in daily communication. It is derived from the ontological system but follows its own principles, optimized for learnability, frequency, and practical use.

**Core axiom:** The word does not carry the structure — it **points** to it. Every word is a projection of an ontological address. Short, usable, anchored in the real world.

---

## The Three-Layer Word Structure

Every nominal word has up to three functional layers:

```
[C₁V₁C₂V₂]  +  [C₃V₃]  +  [C₄V₄...]
   Layer 1          Layer 2       Layer 3+
  Ontological      Category      Identity
   address         (usage)      (distinction)
```

### Layer 1 — Ontological Address (Syllables 1–2, CVCV)

The first two syllables define **what the thing IS** in the ontological system.

- **C₁** = primary domain (MANDATORY — always mirrors the domain)
- **V₁** = row position in the domain matrix (A=essence → U=property)
- **C₂** = secondary domain / operational axis (PREFERENTIAL)
- **V₂** = column position in the domain matrix (A=base → U=specialization)

This address is derived directly from the domain matrices. Every CVCV base points to a unique position in the ontological space.

**Examples:**
- *popa* → P(matter) + O(combination) + P(matter) + A(base) = mixture at the most general level
- *papa* → P(matter) + A(essence) + P(matter) + A(essence) = matter itself
- *pupa* → P(matter) + U(property) + P(matter) + A(base) = physical state/phase

### Layer 2 — Category (Syllable 3, CV)

The third syllable defines **where the thing lives in human use**. This is a pragmatic index — not derived, but assigned from a standardized set.

**Key rule:** The third consonant belongs to the same closed set of 20 consonants. It is a **reindexation of the ontological domains into the space of human use** — same structure, different perspective.

The category is chosen by **dominant function in human use**, not by total composition.

> *"Milk is an aliment first, biological second, liquid third."*

### Layer 3+ — Identity (Syllable 4+, CV...)

Additional syllables are **purely distinctive** — they differentiate specific instances within the same category. No semantic derivation rules apply here. A word at this level is simply a stable, memorable form anchored to its category.

## Projection Rules

| Element | Rule | Notes |
|---|---|---|
| **C₁** | MANDATORY | Always mirrors the primary domain. P-words = matter, N-words = life, M-words = mind. |
| **C₂** | PREFERENTIAL | Soft semantic clustering. Overridable for phonetics or collision avoidance. |
| **V₁, V₂** | FROM MATRIX | Derived from row/column position in the domain matrix. |
| **C₃** | CATEGORY (closed set) | One of 20 consonants, representing the usage domain. |
| **V₃+** | FREE | Purely distinctive within the category. |
| **Word length** | FREQUENCY | Shorter = more common/important. |

---

## Word Length Scale

| Length | Format | Use |
|---|---|---|
| CVCV | 2 syllables | Universal core — highest frequency, most important concepts |
| CVCVCV | 3 syllables | Common — specific but daily concepts |
| CVCVCVCV | 4 syllables | Less common — technical or specialized |
| CVCVCVCVCV+ | 5+ syllables | Rare — highly specific or scientific |

A 2-syllable word is the **word-mother** — the broadest, most productive entry in its semantic field. All longer words in the same field share its first two syllables.

---

## The Semantic Axis Law

> **Meaning = Ontology(C₁) × Operation(C₂)**

C₁ signals the domain. C₂ signals the operational direction within that domain. A speaker who knows the 20 base domains can infer the semantic family of any unfamiliar word from its first two consonants alone.

This law applies to Layers 1 and 2. Layer 3+ is exempt.

## The 20 Usage Categories (Layer 2 — C₃)

The third consonant indexes the word into one of 20 usage categories. These categories are **isomorphic** with the 20 ontological domains — same structure, human-use perspective.

**Official table:**

| C₃ | Usage Category | Scope | Example (from *popa* = mixture) |
|---|---|---|---|
| **-p-** | natural raw substances | unprocessed matter | *popapa* = clay |
| **-b-** | biological / corporeal | living origin, vital functions | *popaba* = blood |
| **-t-** | food and beverages | nutritive consumption | *popata* = soup |
| **-g-** | geological / mineral | inorganic, terrestrial origin | *popaga* = concrete (material) |
| **-r-** | transitory physical states | mud, vapor, gel, foam | *popara* = mud |
| **-l-** | applied physical structures | paste, layer, mass | *popala* = paste |
| **-s-** | granulates and powders | flour, sand, dust | *popasa* = flour mix |
| **-d-** | industrial / manufactured | paint, rubber, plastic | *popada* = paint |
| **-k-** | collective / social use | cement, concrete in use | *popaka* = mortar in use |
| **-n-** | organic / agricultural | compost, humus, resin | *popana* = compost |
| **-v-** | energetic / fuel | oil, gas, coal | *popava* = fuel mixture |
| **-ǰ-** | internal / active fluids | syrup, solvent, glaze, bodily fluid | *popaǰa* = glaze |
| **-x-** | reactive / corrosive | acid, base, oxidizer | *popaxa* = acid |
| **-m-** | medical / biochemical | medicine, serum, antidote | *popama* = liquid medicine |
| **-ľ-** | sensorially marked | perfume, spice, olfactory substance | *popaľa* = perfume |
| **-y-** | marking / pigmentation | pigment, ink, dye | *popaya* = pigment |
| **-w-** | affective / ritual | honey, incense, ritual herb | *popawa* = honey |
| **-z-** | extreme / dangerous | explosive, strong toxin, radioactive | *popaza* = poison |
| **-h-** | environmental / atmospheric | mist, smoke, airborne particle | *popaha* = mist |
| **-f-** | external interface / contact | contact lubricant, adhesive, surface agent | *popafa* = adhesive |

**Disambiguation rule:** When a thing fits multiple categories, choose the **most salient function in human use**. The same physical substance can have different words depending on the perspective:
- *popada* = paint (as manufactured product)
- *popaya* = pigment (as marking/coloring agent)
- *popaǰa* = the fluid itself (internal/active)
- *popafa* = the same fluid as surface agent (external interface)

All are valid. They are different words for different aspects of the same substance.

**Formal definition of C₃:**
> *The third consonant encodes the dominant functional class of human use of the substance, chosen by practical salience and not by complete ontological composition.*

## Relationship Between Ontological and Usage Categories

The usage categories (C₃) are a **functional reindexation** of the ontological domains — not a repetition. They share the same structural skeleton but serve different purposes:

| | Ontological Domain | Usage Category |
|---|---|---|
| Perspective | What the thing IS in reality | Where it lives in human experience |
| Function | Universal, independent of observer | Pragmatic, human-centered |
| Example | N = life (organism, biology) | -b- = biological (blood, milk, tissue) |

They are **isomorphic but not equivalent**. The same 20 consonants index reality from two different angles.

## Word-Mother Principle

Every CVCV word is a **word-mother** — a productive root that generates a family of more specific words through Layer 2 and Layer 3 extensions.

**The word-mother carries the broadest meaning** in its semantic field. Every extension narrows it.

Example — *popa* (mixture):

```
popa          → mixture (word-mother, maximum generality)
popata        → soup (mixture + food category)
popata-[V]    → specific soup (mixture + food + identity vowel)
```

The word-mother is always valid and usable. Speakers who don't know a specific word can always use the word-mother — they will be understood at the right level of generality.

## Extraction Protocol (From Matrix to Word-Mother)

When extracting word-mothers from domain matrices:

1. **Locate the cell** in the domain matrix (row × column intersection)
2. **Read the ontological description** — what is the abstract concept?
3. **Find the simplest real-world anchor** for that concept
4. **Verify domain purity** — the word must remain in its primary domain
5. **Assign the word-mother** — the CVCV form already determined by the matrix

**Validation tests:**
- Does this word describe something that is still clearly in domain P (or T, K, B...)?
- Does it contaminate another domain? (If yes → wrong cell or wrong word)
- Is it general enough to generate a productive family?

**What to avoid:**
- Words that are already too specific for a word-mother (*popada* = paint is Layer 2, not a word-mother)
- Words that belong to another domain (*papo* = system would belong to K or L, not P)
- Words that are processes rather than entities (*popa* = mixing would belong to V, not P)

## Quick Reference — Decision Rules

1. **C₁** always = primary domain. Never override.
2. **C₂** preferentially = operational direction. Override only for phonetics.
3. **Layer 1** (CVCV) = derived from matrix. Every cell = one word-mother.
4. **Layer 2** (C₃) = usage category from the official table of 20.
5. **Layer 2** choice = dominant function in human use, not total composition.
6. **Layer 3+** = free, purely distinctive.
7. **Word-mother** = always valid at its level of generality.
8. When in doubt: use the word-mother, not a derived form.
