## рџ”Ќ What Are Vectors?

**Vectors** are compositional words that start with a vowel and systematically describe concepts through ontological coordinates.

### **Simple Analogy**

**Regular dictionary:**

- Look up "photosynthesis"
- If you don't know the word, you're stuck

2. **Which aspect?** (Origin, Function, Structure, etc.)
3. **Which specific variation?** (25 options per aspect)
4. **What intensity?** (minimal в†’ maximal)

**Result:** Precise, unambiguous concept description.

## рџЏ—пёЏ Vector Structure

### **Four Levels of Specification**

#### **Level 1: Domain (VCV)**

**Format:** Vowel + Consonant + Vowel  
**Examples:** `apa`, `oma`, `inu`

**The initial VCV identifies:**

- Which of the 25 domains
- Which of 5 subcategories within that domain

**Pattern:**

```
a-p-? = Domain P (Substance), Subcategory A (Origin)
e-p-? = Domain P (Substance), Subcategory E (Vitality)
i-p-? = Domain P (Substance), Subcategory I (Complexity)
o-p-? = Domain P (Substance), Subcategory O (Function)
u-p-? = Domain P (Substance), Subcategory U (Taxonomy)
```

---

#### **Level 2: Vector (CV)**

**Format:** + Consonant + Vowel  
**Examples:** `apa-ma`, `oma-ri`, `inu-ke`

**The CV identifies:**

- Which of 25 specific vectors within the subcategory
- What gradation/intensity (a/e/i/o/u)

**Each subcategory has 25 vectors:**

```
Subcategory a-p (Origin):
  Vector 1: p-a/e/i/o/u (natural origin gradations)
  Vector 2: m-a/e/i/o/u (biological origin gradations)
  Vector 3: n-a/e/i/o/u (artificial origin gradations)
  ... (22 more vectors)
  Vector 25: x-a/e/i/o/u (hybrid origin gradations)
```

---

#### **Level 3: Additional Precision (CV+)**

**Format:** + Consonant + Vowel (repeatable)  
**Examples:** `apa-ma-ri`, `oma-ri-ke-fu`

**Optional CV additions provide:**

- Further specification
- Qualifiers
- Contextual markers
- Compound concepts

**You can add as many CV syllables as needed for precision.**

---

#### **Level 4: Gradation (Final Vowel)**

**The FINAL vowel indicates intensity/degree:**

|Final Vowel|Meaning|Example Context|
|---|---|---|
|**-a**|Minimal, negative, absence|`apa-ma` = barely natural, almost artificial|
|**-e**|Weak, low, beginning|`apa-me` = weakly natural, slightly natural|
|**-i**|Neutral, standard, balanced|`apa-mi` = naturally occurring (standard)|
|**-o**|Strong, high, intense|`apa-mo` = strongly natural, very natural|
|**-u**|Maximal, absolute, extreme|`apa-mu` = absolutely natural, purely natural|

**This gradation principle applies UNIVERSALLY across all vectors.**
