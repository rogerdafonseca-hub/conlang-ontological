## [0.1.0] - 2026-01-XX — Initial Public Release

### Added

#### Core System Architecture
- **Phonemic Inventory** — 25 consonants + 5 primary vowels (IPA-based)
- **4-Layer Modular System:**
  - Layer 1: Basic Grammar (CV) — 125 resources
  - Layer 2: Extended Grammar (VCV) — 525 resources
  - Layer 3: Nominal Lexicon (CV+CV+) — Framework established
  - Layer 4: Ontological Vectors (VCV+CV+...) — 62,500+ concept capacity

#### Grammar System (650 Resources)
- **25 Basic Grammatical Families (CV):**
  - Interrogatives (k), Modality (m), Time/Aspect (t), Negation (n), Prepositions (p)
  - Connectives (l), Subordination (s), Demonstratives (d), Quantity (g), Cases (z)
  - Cognitive Verbs (r), Physical Actions (f), Communication (h), Auxiliary Verbs (v)
  - Reflexive (w), Evaluation (y), Spatial Deixis (š), Number/Plural (ň)
  - Special Aspects (č), Comparison (ž), Evidentiality (ǰ), Manner Adverbs (ñ)
  - Determiners (c), Pragmatic Markers (x), Inception/Completion (b)

- **Extended Grammar (VCV):**
  - 25 clusters per family (base + 5 gradation groups)
  - Systematic intensification: negative/minimal (a-) → maximal/absolute (u-)

#### Ontological Matrices (1,100 Concepts Documented)

- **Matrix P (Substance/Being) — COMPLETE (500 concepts)** (to push soon)
  - Subcategories: Origin, Vitality, Complexity, Function, Taxonomy
  - 20 vectors × 5 gradations per subcategory
  - Complete taxonomic classification from fields to artificial constructs

- **Matrix M (Dimensional/Spatial) — COMPLETE (500 concepts)** (to push soon)
  - Subcategories: Dimensionality, Relative Position, Distance/Scale, Direction/Orientation, Form/Topology
  - Covers 0D through 4D+ concepts
  - Comprehensive spatial relationship system

- **Matrix N (Gradual/Temporal) — PARTIAL (100 concepts, 20%)** (to push soon)
  - Subcategory a-n (Duration) — COMPLETE
  - 20 vectors covering instantaneous → eternal timeframes
  - Systematic temporal quantification

#### Design Decisions
- **Universal phonology** — Top 25 most common consonants across world languages
- **Cross-linguistic vocabulary strategy** — Import cognates present in 5+ languages
- **Ontological completeness** — Systematic 25-domain reality categorization
- **Graceful degradation** — Learnable at multiple complexity levels
- **Public domain** — No restrictions on use or modification

### Philosophy Established
- **Complete** — Express any human concept with precision
- **Lightweight** — 30 phonemes, regular grammar, predictable patterns
- **Intuitive** — Based on linguistic universals, not cultural bias
- **Systematic** — Mathematical structure, compositional semantics
- **Universal** — Accessible to speakers of any native language
