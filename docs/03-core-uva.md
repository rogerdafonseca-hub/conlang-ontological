# 📘 UNIVERSAL VOWEL AXIS (UVA)

**Formal Semantic Core of the Language**

---

## 🎯 Definition

The **Universal Vowel Axis (UVA)** is a five-level ordinal system that encodes **semantic intensity, precision, and relational resolution** across all grammatical domains.

It functions as a **unified semantic gradient**, applicable to:

- space
    
- time
    
- causality
    
- epistemology
    
- intention
    
- relation
    
- and all derived grammatical systems
    

---

## ⚙️ Formal Structure

```
UVA = {a, e, i, o, u}

Order: a < e < i < o < u  
Type: closed ordinal scale  
Function: continuous semantic modulation
```

---

## 📊 Core Semantic Values

|Vowel|Level|Core Value|General Meaning|
|---|---|---|---|
|**a**|1|Diffuse|weak, vague, incomplete, peripheral|
|**e**|2|Partial|intermediate, approximate, limited|
|**i**|3|Neutral|standard, balanced, unmarked|
|**o**|4|Direct|strong, specific, targeted|
|**u**|5|Absolute|maximal, total, precise, essential|

## 🔁 Formal Properties

### 1. Monotonic Progression

Meaning always intensifies or converges from left to right:

```
a → e → i → o → u
(diffuse → absolute)
```

---

### 2. Semantic Invariance

Each vowel preserves its **core semantic nature across all domains**:

|Domain|a (Diffuse)|u (Absolute)|
|---|---|---|
|Space|vague location|exact point|
|Time|undefined time|precise instant|
|Causality|weak cause|deterministic cause|
|Knowledge|assumption|certainty|
|Relation|loose|inseparable|

**Key Principle:**  
Vowels do not change meaning — only their domain of application changes.

- structural reference
    
- baseline meaning

### 4. Polar Extremes

## 🔒 Mandatory Rules

### 1. No Axis Inversion

Never invert the semantic scale:

- **a cannot express strong/absolute meaning**
    
- **u cannot express weak/diffuse meaning**
    

---

### 2. Maintain Continuous Gradience

All vowel series must follow:

> less → more

---

### 3. Avoid Semantic Jumps

Incorrect:

- e = "maybe"
    
- i = "certainty"
    

Correct:

- e = partial
    
- i = neutral
    
- o/u = certainty
    

---

### 4. Single Global Axis

There is only **one universal vowel axis**.

No sub-axes or alternative vowel systems are allowed.
