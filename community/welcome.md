## 📚 How It Works

### **Three Progressive Layers**
```
LAYER 1: GRAMMAR (250 particles)
├── CV: Basic grammar (ma = can, ta = past)
├── VC: Extended grammar (am = cannot, im = might)
└── Learn in: 2-4 weeks

LAYER 2: CORE VOCABULARY (625 words)
├── VCV: Everyday words from 25 domains
├── avi (go), ewi (cry), uwi (laugh), ami (place), eno (day)...
└── Learn in: 3-6 months

LAYER 3: TECHNICAL VECTORS (390,625 concepts)
├── VCV+CV+...: Precision terminology (written as one word)
├── Optional — reference as needed
└── Learn in: Never (use like a dictionary)
```

**Total for fluency: 875 resources**  
**Total available: Infinite precision**

## 💡 What Makes This Language Special?

### **1. Systematic, Not Arbitrary**

Every word follows predictable patterns:
```
Domain W (Emotion):
awa = frown (anger, minimal)
awe = complain (anger, weak)
awi = argue (anger, neutral)
awo = shout (anger, strong)
awu = rage (anger, maximal)
```

**Learn the pattern once. Generate infinite words.**

---

### **2. Structure = Meaning**

You can tell what a word is just by looking at it:

| Pattern | Type | Example |
|---------|------|---------|
| CV | Basic grammar | `ma` (can) |
| VC | Extended grammar | `am` (cannot) |
| VCV | Everyday word | `awi` (argue) |
| CV+CV+ | Specific object | `kalu` (dog) |
| VCV+CV+ | Technical term | `apimabi` (biological-active-organism) |

**No confusion. Instant recognition.**

---

### **3. Graceful Scaling**

**Casual speaker** (875 words):
```
ba ewi te
I cry present
"I'm crying"
```

**Technical speaker** (same grammar + precision):
```
ba epimote te pi apimabi
I emotional-state-sad present at organism-biological
"I am experiencing sadness at the biological-organism level"
```

**Same foundation. Choose your precision.**

## 🎨 Quick Examples

### **Basic Conversation**
```
Person A: ka be wa?
          what you want?
          "What do you want?"

Person B: ba wa umi
          I want home
          "I want to go home"

Person A: bo ma aci
          we can move
          "We can go together"
```

### **Express Emotion**
```
ba uwi te    (I laugh present)
be ewi ta    (you cry past)
bi awi lo    (they argue if)
```

### **Ask Questions**
```
ka?    (what?)
ke?    (who?)
ki?    (where?)
ko?    (when?)
ku?    (why?)
```

**Simple. Clear. Effective.**
