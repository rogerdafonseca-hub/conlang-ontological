# Project Roadmap

**Universal Ontological Language Development Plan**

*Living document — Updated regularly as progress is made.*

---

## 🎯 Vision

Build a truly universal constructed language that combines:
- **Ontological completeness** — systematic categorization of all reality (390k concepts)
- **Cross-linguistic accessibility** — phonology and vocabulary from world languages
- **Dual-tier architecture** — accessible Essential Core + precise Professional Expansion
- **Practical usability** — from casual conversation to technical precision

## 🚀 Dual-Tier Development Strategy

### Essential Core (Priority)
**Focus:** Accessibility, learnability, practical use

**Development approach:**
- Document systematically (matrix by matrix)
- Create learning materials in parallel
- Prioritize most useful domains
- Community testing and feedback
- Iterative refinement

**Timeline:** 2026-2027 (12-18 months)

---

### Professional Expansion (Secondary)
**Focus:** Precision, completeness, AI/scientific use

**Development approach:**
- Build on Essential Core foundation
- Prioritize technical/scientific domains
- Community contributions encouraged
- API-first design (programmatic access)
- Integration with existing ontologies (OWL, Schema.org)

**Timeline:** 2027-2028 (18-24 months)
